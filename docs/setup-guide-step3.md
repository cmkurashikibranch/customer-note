# お客様ノート ステップ3 セットアップ手順（AI自動読取）

名刺の写真を Claude（AI）が読み取り、各欄を自動入力する機能のセットアップ手順です。
APIキーはGAS（Google Apps Script）のスクリプトプロパティだけに置き、公開コード・アプリには一切出しません。

```
[スマホのアプリ] --写真--> [あなた専用GAS] --中継--> [Claude AI]
   公開ページ              APIキーはここだけ          名刺を読み取り
```

---

## 1. Anthropic APIキー

1. https://console.anthropic.com でアカウントを作成。
2. APIキーを発行（`sk-ant-...`）。
3. 支払い（クレジット）を設定。少額でOK（Haikuは1枚あたり約0.3円）。

## 2. GAS Web App

1. https://script.google.com で新規プロジェクトを作成。
2. `gas/Code.gs` の内容を**まるごと貼り付け**（※ブラウザの自動翻訳はOFFに。`if` などが翻訳されてコードが壊れるのを防ぐ）。
3. 「プロジェクトの設定」→「スクリプト プロパティ」に2件登録：
   - `ANTHROPIC_API_KEY` = 発行したAPIキー
   - `APP_SECRET` = 任意の合言葉（英数16文字程度。アプリ側にも同じものを入れる）
4. `testProxy()` の `SAMPLE` を実際の名刺写真の dataURL に差し替えて実行 → 実行ログに `{"ok":true,...}` が出ればAPI接続成功。
   - dataURLは、アプリで名刺を撮った後にブラウザのConsoleで `copy(regPhotoData)` を実行すると取得できます。
5. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」→「自分として実行」「アクセスできるユーザー: 全員」でデプロイ → `/exec` で終わるURLを控える。
   - ※コードを直して再デプロイするとURLが変わる/アクセス権が既定に戻ることがあります。変えたらアプリ側の設定を貼り直してください。

## 3. アプリ側の設定

1. アプリ右上の ⚙️ → 設定画面を開く。
2. 「AI読み取りサーバーURL」に `/exec` URL、「合言葉」に `APP_SECRET` と同じ文字列を入力。
3. 「保存して接続確認」→「✅ つながりました」＋「本日あと N 枚」が出ればOK。
4. 人に渡すとき：「設定リンクをコピー」→ そのリンクをQRコードにして見せる。相手はQRを読むだけで設定完了（合言葉の手入力不要）。

## 使い方

1. 登録画面で名刺を撮る → 写真の下に「🤖 AIで読み取る」が出る。
2. 押す → 数秒で各欄が自動入力（名前・会社は「推測」、電話・メールは「要確認」タグ付き）。
3. 内容を確認・修正して保存。
4. うまく撮れない時は、下の「うまく撮れない時：名刺の文字を貼り付けて読み取る」（OCR貼り付け）も使えます。

## 注意・調整

- **1日の上限**：`gas/Code.gs` の `CONFIG.maxPerDay`（既定50枚）で変更。日本時間0時にリセット。
- **モデル**：`CONFIG.model`（既定 `claude-haiku-4-5`）。精度が要るなら `claude-sonnet-4-6` 等に変更（コスト上昇）。
- **画像サイズ上限**：`CONFIG.maxImageBytes`（既定1.5MB）。アプリは長辺800px・JPEG圧縮で送るので通常は収まります。
- **セキュリティ**：APIキーはGASのスクリプトプロパティのみ。URL秘匿＋合言葉＋1日上限＋画像サイズ上限の多層で保護。
- オフライン時はAIボタンを出しません（手入力・OCRは使えます）。
